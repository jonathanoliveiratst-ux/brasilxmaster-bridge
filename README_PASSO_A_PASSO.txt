BRASILXMASTER BRIDGE - SOLUCAO DEFINITIVA PELO GITHUB

Esta versao NAO precisa compilar no seu Windows.
O GitHub gera o .EXE para voce.

PASSO A PASSO:

1. Entre em:
   https://github.com

2. Crie uma conta ou entre na sua.

3. Clique em:
   New repository

4. Nome:
   brasilxmaster-bridge

5. Pode deixar Private.

6. Clique em:
   Create repository

7. Clique em:
   uploading an existing file

8. Envie TODOS os arquivos deste ZIP.
   ATENCAO: envie tambem a pasta oculta .github

9. Clique em:
   Commit changes

10. Clique na aba:
    Actions

11. Clique em:
    Build Windows EXE

12. Clique em:
    Run workflow

13. Quando ficar verde/concluido, clique na execucao.

14. Baixe o arquivo em Artifacts:
    BrasilXMaster-Bridge-Windows-EXE

15. Dentro estará o instalador .EXE.

O APP FAZ:
- Abre https://brasilxmaster.com.br
- Cria http://127.0.0.1:9999
- Responde /detect
- Responde /interfaces
- Detecta J2534
- Detecta RP1210
- Detecta unidades/pendrive
